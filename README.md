# 腾讯文档自动化控件

为了抢志愿写的

实现了定时开始自动填写收集问卷

基本上可以在1s内完成问卷填写

### 使用教程：
1、先配置好python环境

2、下载与自己浏览器版本匹配的浏览器驱动

3、相关依赖库使用pip install下载

4、在本文件夹目录下打开命令行窗口，执行：
```
python main.py
```
5、按提示操作
